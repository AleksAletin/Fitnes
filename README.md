# Подопечные

Нативное iOS-приложение для персонального тренера: клиенты, пакеты занятий и расписание
в одном месте. Заменяет связку «Календарь айфона + Заметки + учёт оплат в голове».

Пользователь — один тренер. Клиентского доступа нет, аккаунтов и сервера нет — всё локально.

## Возможности (V1)

- **Сегодня** — расписание дня с календарём в шапке (неделя↔месяц, точки тренировок,
  плавающие выходные). Свайп: провести / перенести / отменить. Тосты с откатом.
- **Клиенты** — список с семафором остатка (🟢/🟡/🔴), сегменты, сортировка, поиск.
- **Карточка клиента** — кольцо пакета (остаток/долг), звонок и Telegram, заметки,
  история занятий и оплат, архив.
- **Гибкие пакеты** — пакет на N занятий / разовое / пробное / абонемент. Учёт оплат.
- **Программа занятия** — упражнения (вес, схема подходов) и свободная заметка.
- **Календарь айфона** — занятия пишутся в отдельный календарь «Тренировки» (EventKit),
  проверка конфликтов по времени.
- **Настройки** — порог жёлтого остатка, окно поздней отмены, пакет по умолчанию.

## Стек

SwiftUI · SwiftData · EventKit · iOS 17+

## Сборка

```bash
xcodebuild build -project Podopechnye.xcodeproj -scheme Podopechnye \
  -sdk iphonesimulator -destination "platform=iOS Simulator,name=iPhone 17"
```

Или открыть `Podopechnye.xcodeproj` в Xcode и запустить на симуляторе/устройстве.

## Структура

```
Podopechnye/
├── Models/      — Client, Package, Lesson, Exercise, Payment, enums
├── Services/    — CalendarService (EventKit), SettingsStore, ToastCenter
├── Views/       — экраны и компоненты
│   └── Forms/   — формы клиента, пакета, занятия, программы
├── Design/      — дизайн-токены (цвета)
└── Resources/   — Assets (иконка, акцентный цвет)
```
