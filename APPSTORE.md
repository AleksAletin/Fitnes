**Description:**
```
«Подопечные» — простой инструмент для персонального тренера. Заменяет разрозненные
заметки и календарь: клиенты, оплаченные пакеты занятий и расписание — в одном месте.

Главное — всегда видно, у кого сколько занятий осталось и кому пора напомнить об оплате.

Возможности:
• Карточки клиентов: контакты, цель, заметки, история занятий и оплат
• Гибкие пакеты: на N занятий, разовое, пробное, абонемент
• Цветовой сигнал остатка: зелёный / жёлтый / красный — сразу ясно, кому пора платить
• Расписание с календарём: неделя и месяц, свайп между периодами, выходные
• Отметка «проведено» одним свайпом — занятие списывается из пакета
• Перенос и отмена занятий, учёт «в долг»
• Программы тренировок: упражнения с весами или свободная заметка
• Интеграция с Календарём айфона: занятия попадают в отдельный календарь «Тренировки»
• Тёмная тема

Все данные хранятся локально на устройстве. Никаких аккаунтов и сбора данных.
```

## Приватность (App Privacy)
- **Data Collection:** **Data Not Collected** — приложение не собирает никаких данных,
  всё хранится только на устройстве, сети и аналитики нет.
- **Privacy Policy URL** (обязательно): захостить `PRIVACY.md` — проще всего включить
  GitHub Pages в репозитории Fitnes, либо дать raw-ссылку на файл.
- **Support URL** (обязательно): можно указать страницу репозитория
  `https://github.com/AleksAletin/Fitnes` или отдельный email/лендинг.

## Что делаешь ты в App Store Connect
1. My Apps → «+» → New App → выбрать bundle id `com.aletin.Podopechnye`, имя, язык RU.
2. Заполнить поля выше (название, описание, ключевые слова, приватность, support/policy URL).
3. Загрузить скриншоты (дам файлы).
4. Age Rating → 4+.
5. Прикрепить билд (загрузим архивом), выбрать его в версии → Submit for Review.

## Что делаю я (когда паковать релиз)
- Соберу **архив для дистрибуции** (Release, distribution-подпись).
- Залью билд в App Store Connect через App Store Connect API key (Issuer ID + Key ID + .p8).
- Подготовлю скриншоты.
